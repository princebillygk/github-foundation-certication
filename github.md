# Github file and directory
1) We can create a new file or upload an file
2) We can also create a directory by creating adding a file with path name
3) We can also edit and delete a file (both require new commit message)
4) We cannot create a empty directory without a file
## Steps on how to upload a file in a directory
1) Go to that directory from github code
2) Add file -> Upload files
3) Upload the file
4) Add commit message 
5) Commit Chages
